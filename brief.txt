Project: Break Out Game
Description: A modern, high-performance web-based Breakout game built with React and HTML5 Canvas. The game features interactive gameplay, a progressive leveling system, and advanced visual effects.

Core Features:
1. Real-Time Physics: Precise ball-to-paddle and ball-to-brick collision detection using coordinate-based logic on a 2D canvas.
2. Dynamic Leveling: Increasing difficulty with each level, including layout changes and score tracking.
3. Enhanced Visuals:
   - High-quality effects managed by a dedicated EffectsManager.
   - Dynamic ball trails and particle-based explosions upon brick destruction.
   - Screen shake animations to enhance the impact of collisions.
   - Glow effects and gradients for a premium visual feel.
4. Responsive Design: Intelligent canvas scaling that adapts to various screen sizes.
5. Gameplay Mechanics: Mouse-controlled paddle for responsive and intuitive player interaction.

Technical Stack:
- Frontend: React (Functional components with Hooks).
- Graphics: HTML5 Canvas API for high-performance 2D rendering.
- Programming Language: JavaScript (ES6+).
- Deployment: Fully compatible with modern hosting services like Vercel and GitHub Pages.

Project Structure:
- src/components/games/breakout/: Contains the core game logic and UI components.
- Board.js: Orchestrates the game loop and rendering.
- EffectsManager.js: Centralized system for particles, trails, and camera effects.
- data.js: Central configuration for game entities (ball, paddle, bricks, player).
